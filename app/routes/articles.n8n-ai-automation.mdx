# Workflow Automation with n8n & Generative AI

Managing consistent high-quality content production across multiple social media platforms demands automated workflow orchestration.

During my AI Automation Internship at **Summit Wealth Builders LLC**, I built end-to-end automation pipelines connecting **n8n**, **Claude**, **Gemini**, and **Make.com**.

## Pipeline Architecture

1. **Content Ingestion**: Fetches market trends and data feeds automatically.
2. **LLM Prompt Engineering**: Synthesizes insights using tailored system prompts with Claude & Gemini APIs.
3. **Image Generation**: Triggers image generation pipelines via DALL-E & Groq APIs.
4. **Automated Publishing**: Schedules and posts verified campaigns directly to LinkedIn.
